对于碳基的吸波材料来说， 决定其吸收特性参数主要包括复介电常数、材料 的厚度和形状等等。对于吸波材料性吸收能的预测， 常用的分析方法有： 等效传 输线法， 传输矩阵法、电路模拟法与跟踪计算法等等[6]。随着商业电磁计算软件 的发展[7] ，可以采用全波仿真的方法来进行评估， 对于吸波材料的设计带来了更 准更快的评估手段。为了章节的相对完备性，我们首先介绍一下基本理论知识。
5.2.1. 吸波材料能量守恒原则
电磁波入射到任意形状的吸波体时候， 都会发生反射、透射、吸收和干涉等 等， 这些综合因素决定了吸波材料的性能， 如吸收峰的位置、宽度和强度。按照能能量守恒的原理， 我们可以把整个吸波材料作为观察对象， 整个体积中能量的 流入、流出关系通过吸波体的外面表面来观察。这样， 可以电磁波在吸波体内传
播的复杂路径问题，简化计算。可以用下面的公式来进行描述：
Pi 一 Pr 一Pt = Pa                             (5. 1)
其中Pi为入射电磁波功率； Pr反射电磁波功率； Pt为透射电磁波功率； Pa为吸波体 吸收的电磁波功率。
那么可以定义功率的反射系数 T = PT/Pi，功率透射系数 t = Pt/Pi，功率
吸收系数a = Pa/Pi。将式(5.1)的两边除以Pi可以得出:
T + t + a = 1                          (5.2)
吸波材料测试时， 我们会在吸波材料的背部加上金属反射板， 可以认为电磁 波没有透射，那么， t=0，r= 1 一 a，因此可以通过反射率系数 r 来判断吸波材 料的吸收性能的优劣。
为了数值研究的方便，亦可以用分贝数(dB)来表示此类系数：
如果用功率计算： T = 10log (dB)，t = 10log (dB)a = 10log(dB)；
用电场值计算:T = 20log  (dB),t = 20log  (dB), a = 20log (dB)(这 里的吸收系数a和电磁波在介质中的衰减系数是不同的)。一般来说， 在有限厚度的吸波材料表面 (非半无限大) ，其电磁波的反射(进
入吸波材料内部后出来的反射)与入射波的相位相反时候，就会产生干涉现象。
当垂直入射进入厚度为 d 的吸波材料后， 入射波与反射波路程相差大于等于 2d，在四分之波长的奇数倍时候有电磁能量相消现象d = (2n + 1)入/4 (n=0，1 ， 2 ，3...) ，由于电磁波在吸波材料内部传输时与材料的电磁参数相关， 因此可以 写成：
d = (2n + 1)c/(4f((e′u′))                    (5.7) 其中， c 为光速,e ′ 、u′为相对介电常数和磁导率， f 为频率。在设计和寻找吸波材 料的吸收峰时候。对于单纯介电损耗的材料，式(5.7)可以写成：
f((e′) = (2n + 1)c/4d                  (5.8)
等式左边介电常数为频率函数为变量， 等式的右边取值与 d 与 n 相关。当 d 确定时， 不同的 n 值对应不同频点的干涉抵消作用； 当 n≥3 的时， 会出现多频率 的干涉消除模式。因此厚度高的吸波材料，会出现多个频段的吸收峰。
